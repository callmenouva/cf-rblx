Privacy

On this page you will find all information regarding privacy management and data handling for the Corsica Ferries RBLX project.

1. Fan-made Project

Corsica Ferries RBLX is an unofficial project not affiliated with Forship S.p.A. / Corsica Ferries – Sardinia Ferries. All content is created solely for entertainment and simulation purposes on Roblox.

2. Automatically Collected Data

During website use, non-identifying technical information may be collected automatically, such as:

  1. anonymized IP address
  2. browser and device type
  3. technical data required for site functionality

This data does not allow personal identification and is used exclusively for security and site functionality purposes.

3. No Accounts and No Forms

The website does not require registration, does not create accounts, and does not collect personal information through forms or contact modules.

4. Cookies

This website does not use tracking or profiling cookies. Technical cookies may be present due to external services such as:

  1. Roblox (for links and embeds)
  2. Discord (for external connections)
  3. Google Fonts (for loading the Montserrat font)

5. External Services

The site uses or links to external services which may collect data according to their own privacy policies:

  1. Roblox Corporation
  2. Discord Inc.
  3. GitHub Pages (site hosting)
  4. Firebase (used only for internal admin panel features)

6. Data Processing

We do not collect, store, or share any personal user data. All technical data is automatically handled by the services that host the site.

7. User Rights

Users may request information regarding any technical data processed by external hosting services (GitHub Pages, Discord, Roblox). The Corsica Ferries RBLX project does not directly manage any database containing personal user data.

8. Changes to the Privacy Policy

This Privacy Policy may be updated at any time without notice. The most recent version will always be available on this page.

9. Contacts

For requests or reports, please use the contact page.
